Git Installation Guide for Linux

1. Open a terminal on your Linux system.
2. Update the package list:
   - **Debian/Ubuntu**: Run `sudo apt update`.
   - **Fedora**: Run `sudo dnf update`.
3. Install Git:
   - **Debian/Ubuntu**: Run `sudo apt install git`.
   - **Fedora**: Run `sudo dnf install git`.
   - **Arch**: Run `sudo pacman -S git`.
5. Verify the installation by running `git --version` in the terminal.
6. If the version displays correctly, Git is successfully installed.
7. Optionally, configure your name and email with:
   - `git config --global user.name "Your Name"`
   - `git config --global user.email "your.email@example.com"`.
