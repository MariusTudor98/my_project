Git Installation Guide for Linux

1. Open a terminal on your Linux system.
2. For **Debian/Ubuntu**: Update the package list with `sudo apt update`.
3. For **Fedora**: Update the package list with `sudo dnf update`.
4. For **Debian/Ubuntu**: Install Git with `sudo apt install git`.5. Verify the installation by running `git --version` in the terminal.
6. If the version displays correctly, Git is successfully installed.
7. Optionally, configure your name and email with:
   - `git config --global user.name "Your Name"`
   - `git config --global user.email "your.email@example.com"`.
